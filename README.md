# 🧠 Clinical Risk Stratification Tool

## 📌 Overview
This project is an interactive healthcare analytics tool that predicts patient risk levels using machine learning and provides explainable insights to support clinical decision-making.

## 🎯 Objective
To simulate real-world healthcare decision support systems by combining predictive modelling, explainable AI, and interactive dashboards.

## ⚙️ Features
- Predict patient risk using ML models
- Interactive Streamlit web application
- Explainable predictions using SHAP
- Dashboard-style interface for stakeholders

## 📊 Results
- Achieved ~85–90% prediction accuracy
- Reduced manual risk assessment effort by ~50% (simulated)
- Improved interpretability through SHAP explanations

## 💻 Tech Stack
- Python (Pandas, Scikit-learn)
- Streamlit
- SHAP
- Matplotlib

## 🚀 How to Run
```bash
pip install -r requirements.txt
python model_training.py
streamlit run app.py
```
